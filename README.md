# Lakshmi Srujana - 3D Portfolio Website

A modern, interactive 3D portfolio website showcasing Lakshmi Srujana's journey as a Computer Science student, IEEE leader, hackathon participant, and campus builder.

## 🌟 Overview

This portfolio features a sleek 3D design with smooth animations, responsive layout, and sections highlighting academic achievements, leadership roles, hackathon experiences, certifications, awards, and contact information.

## ✨ Features

- **3D Hero Section**: Interactive canvas background with dynamic visuals
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: CSS transitions and JavaScript-powered interactions
- **Navigation**: Sticky navbar with mobile hamburger menu
- **Sections Include**:
  - About: Personal introduction and background
  - Student Life: Academic journey and current focus
  - Leadership: IEEE and campus initiatives
  - Hackathons: Competition experiences and achievements
  - Certifications: Professional credentials (20+)
  - Awards: Recognition and scholarships
  - Contact: Ways to get in touch

## 🛠️ Tech Stack

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom properties, gradients, animations, and responsive design
- **JavaScript**: Interactive elements and canvas animations
- **Fonts**: Google Fonts (Outfit and JetBrains Mono)

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely in the browser

### Installation

1. Clone or download this repository
2. Navigate to the project directory
3. Open `index.html` in your web browser

```bash
# Example commands (if using a local server)
cd "3D PORTFOLIO WEBSITE"
python -m http.server 8000  # Python 3
# or
npx serve .  # If you have Node.js installed
```

Then open `http://localhost:8000` in your browser.

## 📱 Usage

- **Navigation**: Use the navbar links to jump to different sections
- **Mobile**: Tap the hamburger menu for mobile navigation
- **Scroll**: Explore content by scrolling through sections
- **Contact**: Use the contact section to reach out
