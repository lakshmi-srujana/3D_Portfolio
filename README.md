# 3D Portfolio - Lakshmi Srujana

A stunning, futuristic 3D portfolio website built with **Three.js**, **GSAP**, and **Vanilla CSS**. This portfolio showcases Lakshmi Srujana's journey as a CS student, IEEE Chair, and innovator.

## 🚀 How to Run

Because the website uses **Three.js** and some advanced browser features, it's best viewed through a local development server.

### Option 1: VS Code Live Server (Recommended)
1. Open the project folder in **VS Code**.
2. Install the **Live Server** extension by Ritwick Dey.
3. Click **"Go Live"** at the bottom right of the editor.

### Option 2: Python (Quickest)
If you have Python installed, run this command in your terminal from the project directory:
```bash
python -m http.server 5500
```
Then visit: `http://localhost:5500`

### Option 3: Node.js / npm
If you have Node installed, you can use `serve`:
```bash
npx serve .
```
Then visit the URL provided in the terminal.

## ✨ Features
- **3D Hero Space**: Interactive particle universe that reacts to mouse movement.
- **Glassmorphism UI**: Modern, premium dark theme with translucent cards.
- **Scroll Animations**: Smooth reveal effects for every section.
- **Interactive 3D Cards**: Hover over cards to see dynamic 3D tilt effects.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **Performance Optimized**: Uses CDNs for Three.js and lightweight assets.

## 📂 Project Structure
- `index.html`: Main structure and content (7 detailed sections).
- `styles.css`: Dark-themed styling, animations, and the design system.
- `main.js`: Three.js logic, particle systems, and interactivity components.
